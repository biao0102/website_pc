游戏介绍——星宿

星宿介绍

1.      星宿模块中共有10种不同的星系，不同的星系对应不同的属星。

2.      星宿模块功能则在玩家等级到达23级开启；

3.      随之玩家等级的不断提升，星宿的中的个个星系逐步开启；

星系属性及开启等级如下：

 

n  紫薇星：气血值等级，26级开启；

n  太阴星：暴击等级，30级开启；

n  贪狼星：普通攻击，23级开启；

n  破军星：法术攻击，23级开启；

n  七杀星：命中，30级开启；

n  武曲星：普通防御，26级开启；

n  天机星：法术防御，26级开启；

n  太阳星：闪避，30级开启；

n  天府星：韧性，30级开启；

点亮星宿

1.      玩家可以通过界面中选择自己中意的星系提升等级；

2.      每个星系中包含多个子星，通过点亮子星来实现星系升级；

3.      在一个星系中，每次点亮10个子星，会额外获得大量属性加成；

4.      选中星系后子星上会有关于这个子星的点亮条件；

5.      每个星系都有着自己独特的行星轨迹。

星宿升级

1.      玩家等级达到点亮条件，等级高于当前选中的子星才可点亮；

2.      点亮子星消耗对应的星力点数，获得足够多的星力才可不断点亮子星，升级星系等级；

3.      升级星系的过程中可能会出现点亮失败，需要玩家一些运气成份。

 

星力的获取

1.      副本通关奖励，任务奖励，星力召唤等；

2.      其中星力召唤可获得大量星力，每日可召唤5次的机会。
