#版本前瞻#帮会挑战赛系统
2015-05-21 11:28:41
0

参赛规则:

n  帮派等级≥2，并且帮派人数≥20人

n  每周5,6,7日18:00~24:00结束

n  每周1的01:00结算


匹配机制：

n  每周第一天所有的满足条件的帮派随机匹配；

n  后面天数就按照胜率来匹配（一胜一负，两胜，零胜两负…）

n  如果匹配完是单数，有剩余的帮派，则扩大胜率范围，在+1或-1的范围查找

n  如果还没有，就再扩大范围

n  仍没有的情况下推送玩家直接胜利，当天不需要战斗，后一天直接匹配


参赛规则：

n  玩家进入地图后看到对方帮派的所有帮众，玩家可以自己选择打那个电脑

n  每个对手都有三个难度级别可以选择对战

u  一星：80%的真实数据

u  二星：100%的真实数据

u  三星：150%的真实数据

n  敌人是AI控制

n  被打过的星级敌人不能再打，同时低于被打过的星级的该对手也不能再次被打（例如，玩家a击败了二星的对手b，则该对手B的一星和二星都不能被挑战了）

n  战胜1星玩家获得1个积分，2星玩家获得2个积分，3星玩家获得3个积分，失败不获得积分

n  挑战赛采取比分制，比赛结束最终积分高的帮派获胜

n  若轮空则直接胜出

n  每个玩家每天只有2次挑战的机会，不能由其他方式增加次数、

n  战斗中退出，直接失败


结算奖励

n  单场战斗奖励

u  选择的星级不同，战斗胜利获得的奖励不同，星级越高奖励越好（帮贡和金币）

u  若是失败，无论星级，同样的奖励（帮贡和金币）

u  若该帮派被轮空，则奖励给参加者战斗2星奖励*2

n  帮派积分总结算

u  获胜的帮派获得帮派资金

u  失败的帮派获得部分帮派资金

u  帮战奖励直接加到帮派资金中

u  结算结果后通过邮件告知所有帮派成员（只胜利方）

u  结算结果在帮派频道中广播：



  胜方：“本帮以300:126的比分战胜了XXX帮派，帮派资金增加XXXXX”

  败方：“本帮以100:126的比分败给了XXX帮派，帮派资金增加XXXXX”

  平局：“本帮以100:100的比分和XXX帮派握手言合，帮派资金增加XXXXX”

l  0:0平局：“本帮和XXX帮派的比分0:0，均未参与挑战赛，帮派资金增加0。请下次积极参加，为帮派做贡献”

u  若两个帮派最终比分战平（不包括0:0），则判为平局，给予奖励与失败奖励相同

u  若两个帮派最终比分为0:0，则视为均没有参加，不给于帮派奖励
