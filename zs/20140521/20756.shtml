升级帮会

2014-05-21 16:31:40分享到：



【帮会等级】1至10级

【升级需要】
帮会资金（由帮众捐献游戏币，帮会任务和特定道具累积帮会资金）
●1级→2级（帮会资金200W）
●2级→3级 (帮会资金消耗750W )
●3级→4级 (帮会资金消耗1800W )
●4级→5级 (帮会资金消耗4200W )
●5级→6级 (帮会资金消耗8000W )
●6级→7级 (帮会资金消耗13500W )
●7级→8级 (帮会资金消耗35000W )

说明：
满足帮会升级条件时“升级”键可操作，否则为单击无效果状态
帮会资金值达到当前等级最大值，必须由帮主将帮会升级
帮会资金达到当前等级最大值，不可进行捐献
每日可捐献1次，分为铜钱捐献和元宝捐献，共用次数
